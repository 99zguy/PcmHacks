Read This First

There is a known issue that can leave your vehicle temporarily unusable. When you turn the key, the engine will turn over but will not start. If this happens, remove the ignition key, remove the fuse (or fuses) that provide power to the PCM, wait 10 seconds, and replace the fuses. Turn on the ignition, wait another 15 seconds, and your car should start right up.

You should familiarize yourself with the location of these fuses before you attempt to do a full read or full write of your PCM. That way if you run into this, you'll already know what to do.

We expect to have this problem fixed before we release the next version.

Supported Devices

This app currently supports the following types of devices:

ObdDiag AllPro - Only the USB and dev-board versions are recommended at this time. The AllPro devices are quite inexpensive, and the USB version is likely to be the most popular choice.

ObdDiag has been a great partner in this project, incorporating changes and responding to feedback. However this means that AllPro devices purchased before early June 2018 will need a firmware update to version 1.21 or later. Updating the firmware on the AllPro adapter requires the FlashMagic tool. Instructions are here and a link to the firmware can be found here.)

Scantool SX or MX - These are inexpensive and superior to ELM knock-off devices, however they do not support 4x (high speed) mode, so a full read takes a little over 20 minutes. Other devices based on Scantool's interface chip will probably work just as well (and just as slowly) but we have not tested any ourselves.

AVT 852 - This is the most expensive option, but this was one of the first interfaces to properly support J1850 VPW high-speed mode.

Any J2534-compatible interface that supports J1850 VPW - At least in theory, these should all work. There are a few of those on the market, and we cannot test them all, but if you have one, give it a try. As time goes by we'll list the ones that are proven to work. The genuine GM MDIs are one device which is known to work, though it is not cheap. Over time we hope to confirm that other, less expensive devices will also work.

Basic Operations

Before you do anything else, you'll need to tell the app what kind of interface device you have, and how it is connected to your computer.

  1. Click the "Select Device" button on the main window - a dialog box will appear.
  2. Select "Serial Port Device" or "J2534 Device" depending on which type you have. If you don't know which, it's probably a serial port.
  3. If you have a serial device, you'll need to select the COM port and device type from the drop-down lists. If you have a J2534 device, you'll need to select the device type.
  4. You can use the Test button to see if you got it right, or just click OK.
  5. Either way, check the Results tab in the main window - it will either print an error message or show some informationa about the selected device.

To read the VIN, serial number, and other properties from the PCM, click the "Read Properties" button. You should generally do this before doing anything else, just to confirm that the device is able to communicate with the PCM.

The "Read Full Contents" button will save the entire contents of the PCM's read-only memory, and it will then ask you where to save that data.

To use the "Modify VIN" feature, click the button and enter your vehicle's VIN, then click OK. Note that the change will not take effect until after you turn off the ignition. The PCM will still run on battery power for a few seconds while it stores the new VIN permanently. If you disconnect all power, the PCM will still have the original VIN the next time you turn it on.

You may find this feature useful if you've purchased a used PCM from a junkyard or online retailer, for example after bricking your PCM with software written by amateurs and downloaded from the internet. (Although, honestly, zero PCMs have been harmed during the development of this app. So far.)

You may notice that there is a button to write changes to the PCM's read-only memory. No, that button doesn't work yet. Yes, we're working on it.

Contributing

The open source portions of this software are distributed at https://github.com/LegacyNsfw/PcmHacks, and the authors welcome contributions.

The reflashing kernel is closed source, due to unfortunate events (and unethical behavior) that have happened in the past. The GitHub repository therefore contains a kernel that can only read from the PCM. However we will create binary releases with the full-functional kernel whenever there are noteworthy improvements to the app or the kernel.
